# Submission Draft

## Description

DuoQuest reimagines Duolingo's website as a retro arcade RPG. The original product stays intact: free bite-sized lessons, 40+ language courses, instant feedback, rewards, personalized learning, plus Math, Music, and Chess. The redesign changes the framing completely: learners enter a quest hub with XP, streaks, world nodes, boss lessons, Story portals, and DuoRadio-style audio moments.

ElevenLabs powers the voice layer. The prototype includes an `/api/narrate` endpoint for Text-to-Speech narration and a browser fallback for local demos. In the final version, each quest node can speak in a character voice, with guide options like Coach, Bard, and Rival for lesson intros, feedback, and conversational navigation.
